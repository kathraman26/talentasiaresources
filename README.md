# Talent Asia Resources Website

Static HTML/CSS/JavaScript website for GitHub Pages.

## Upload
1. Extract the ZIP.
2. Open the `TalentAsiaResources_Final` folder.
3. Upload all files and folders inside it to the root of the GitHub repository.
4. In GitHub: Settings → Pages → Deploy from branch → `main` → `/ (root)`.
5. The included `CNAME` file points GitHub Pages to `www.talentasiaresources.com`.
